## 📋 Sobre o Projeto

O **Gerador de Nomes Piratas** é uma aplicação web totalmente gratuita que cria nomes épicos de piratas com mais de **27.000 combinações únicas**. Ideal para RPG, videogames, cosplay, fanfiction, festas temáticas ou simplesmente para se divertir nos Sete Mares.

## ✨ Funcionalidades

- 🎲 **Geração aleatória** — mais de 27.000 combinações únicas de nomes
- ⚧ **Seleção de gênero** — nomes masculinos e femininos
- 🌍 **Multilíngue** — suporte a Português 🇧🇷, Inglês 🇺🇸 e Espanhol 🇪🇸
- 🗺️ **Quiz de personalidade** — descubra se você é ⚔️ Agressivo, 🧠 Estratégico ou 🌑 Misterioso
- 💡 **Sugestões extras** — outros nomes que você pode gostar a cada geração
- ⭐ **Sistema de favoritos** — salve os nomes que mais te agradaram
- 📋 **Copiar com 1 clique** — copie o nome gerado facilmente
- 📤 **Compartilhamento social** — envie pelo WhatsApp, Instagram, TikTok e mais
- 📱 **Design responsivo** — funciona perfeitamente em desktop, tablet e celular

## 📋 About the Project

The **Pirate Name Generator** is a completely free web application that creates epic pirate names with over **27,000 unique combinations**. Perfect for RPG, video games, cosplay, fanfiction, themed parties, or simply having fun on the Seven Seas.

## ✨ Features

- 🎲 **Random generation** — over 27,000 unique name combinations
- ⚧ **Gender selection** — masculine and feminine names
- 🌍 **Multilingual** — supports Portuguese 🇧🇷, English 🇺🇸, and Spanish 🇪🇸
- 🗺️ **Personality quiz** — discover if you are ⚔️ Aggressive, 🧠 Strategic, or 🌑 Mysterious
- 💡 **Extra suggestions** — other names you might like on each generation
- ⭐ **Favorites system** — save your favorite generated names
- 📋 **One-click copy** — easily copy the generated name
- 📤 **Social sharing** — share on WhatsApp, Instagram, TikTok and more
- 📱 **Responsive design** — works perfectly on desktop, tablet and mobile
